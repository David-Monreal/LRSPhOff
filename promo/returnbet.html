RETURN BET

Ang Return Bet (RB) ay isang espesyal na pagkakataon sa pagtaya na magbibigay sa iyo ng isa pang pagkakataon upang itaya ang iyong mga numero para sa susunod na draw.

Ang mga Pick 3 Games ay may kasamang Return Bet (RB) para sa mga taya na 50 pesos pataas. Ito ay kailangang gamitin agad sa susunod na draw!

Paano makakuha ng Return Bet (RB)?

Pagtaya sa Pick 3 Games
Sa mga Pick 3 games tulad ng 6/45, 6/49, 6/55, at 6/58 Lotto, pipili ka ng tatlong numero mula sa isang set ng mga numero.

Mga halimbawa:

  • 6/45 Mega Lotto ‚û§ 1 - 45
  • 6/49 Super Lotto ‚û§ 1 - 49
  • 6/55 Grand Lotto ‚û§ 1 - 55
  • 6/58 Ultra Lotto ‚û§ 1 - 58

 

Dalwang Numero
Ang RB ay natatanggap kapag may dalawang tamang numero mula sa tatlong napili o taya ang lumabas mula sa anim na bola ng lotto draw.

Halimbawa:
Ang lumabas na anim na numero sa lotto draw ay 1-2-3-4-5-6

Bet Amount
Ang Return Bet ay maaari lamang makuha kung ang halaga ng iyong taya ay 50 pesos pataas. Kaya, kung ang taya mo ay mas mababa sa 50 pesos, hindi ka kwalipikado para sa Return Bet.

Listahan ng mga Return Bet

Maglalabas ng opisyal na listahan ng mga bettor na nakakuha ng RB. Ang listahang ito ay ipapadala sa group chat kung saan ka tumaya upang matiyak na alam ng lahat kung sino ang may RB at maaari mo rin itong tingnan sa‚ñ∫ LIST OF RETURN BETS.

 

Validity
Ang Return Bet ay kailangang gamitin sa susunod na araw at hindi ito pwedeng itaya kung lumipas na ang araw ng validity.

 

Paghahati ng Taya

Ang RB ay maaaring hatiin sa maliliit na halaga ng taya, ngunit ang bawat taya ay dapat hindi bababa sa 50 pesos. Halimbawa, kung ang RB amount mo ay 500 pesos, maaari itong hatiin sa limang taya na tig-100 pesos o sampung taya na tig-50 pesos.

Note:
Mahalagang tandaan na ang bawat hati ng taya ay kailangang hindi bababa sa 50 pesos. Hindi maaaring bumaba sa halagang ito upang makasunod sa patakaran ng paggamit ng RB.

 

Paano Itaya ang Return Bet?

Kapag meron kang Return Bet, kailangan mong maglagay ng bagong taya sa susunod na draw, at gamitin ang Return Bet bilang bahagi ng iyong bagong taya.
