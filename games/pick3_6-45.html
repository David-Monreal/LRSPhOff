Pick 3 Lotto

Subukan ang iyong swerte sa 6/45 Mega Lotto! Pumili ng 3 numero mula 1 hanggang 45.

Tuwing Lunes at Miyerkules ang draw ng 6/45 Mega Lotto, kaya may dalawang (2) beses kang pagkakataon bawat linggo na manalo.

 

PAANO TUMAYA?

1. Pagpili ng Numero
Sa Pick 3 game na 6/45 Mega Lotto, ikaw ay pipili ng 3 numero mula 1 hanggang 45. Ang mga napili mong numero ay ang magiging kombinasyon ng iyong taya.

 

2. Paglalagay ng Bet Amount
Ang iyong taya ay kailangang umabot sa minimum na halaga ng ₱25. Maaari mong dagdagan ito, basta't ito ay divisible by 5 (halimbawa, ₱30, ₱35, ₱40, ₱45, at iba pa).

 

3. Format sa Pagtaya
Siguraduhing tama ang format sa pagtaya para ma-validate ang inyong taya.

Mga dapat tandaan:
📌 Lagyan ng "dash" ( - ) ang pagitan ng bawat numerong taya.
📌 Maglagay ng "equal sign" ( = ) pagkatapos ng mga numerong napili at ilagay ang bet amount sa dulo.
📌 Kung ang taya ay kabilang sa Promo Bet, ilagay sa dulo ng taya ang code kung saan ito na bibilang.
➤Kung ang taya ay parte ng FB o Freebet, huwag na maglalagay ng bet amount
➤Kung ito naman ay RB o return bet, kailangang ilagay ang bet amount at ang code na RB sa dulo

FORMAT:
Regular Bet:
❌ 1,2,3,25
❌ 1 2 3 25
✔️ 1-2-3=25
 
FreeBet (FB):
❌ 1-2-3=25FB
✔️ 1-2-3=FB
 
Return Bet (RB):
❌ 1-2-3=RB50
✔️ 1-2-3=50RB

 

4. Pag-send ng Taya

Ipadala ang iyong napiling numero kasama ang bet amount sa Messenger Group Chat. Kasama rin dito ang pagsasagawa ng bayad gamit ang napagkasunduang paraan, tulad ng GCash, PayMaya o bank transfer.

Note:

  • Tandaan na kapag hindi na-validate (walang react o walang kumpirmasyon) ang inyong taya ay hindi ito isasama sa listahan ng mga maaring manalo.
  • Pagkatapos i-validate ng Admin and inyong taya ay kailangan itong i-send sa LRS Official Encoding GC, hintayin ang kumpirmasyon ng GC Admin para makasiguradong pumasok ang inyong mga taya.

 

PAANO MANALO?

Kapag ang lahat ng 3 numero na iyong napili ay lumabas sa anim na bolang nabunot sa 6/45 Mega Lotto draw, ikaw ay itinuturing na panalo at makakatanggap ng premyo depende sa itinalagang halaga ng jackpot.

 

Halimbawa:
Ang iyong taya ay 12-45-23 tapos ang lumabas na numero PCSO Live Draw ay ang mga numerong nasa ibaba, ikaw ay siguradong panalo, kahit na magkabali-baliktad pa ito.
